# Grand Line Runner
Generated by [Rojo](https://github.com/rojo-rbx/rojo) 7.6.1.

## Getting Started
This repository is the source of truth for scripts and Rojo-managed instances. The live place still depends on unmanaged world assets and templates that are not fully represented under `src` yet, so a clean `rojo build` is a scripted baseline, not a complete production place rebuild.

Use Rojo 7.6.1 via either `rokit` or `aftman`.

To build the current Rojo-managed project, use:

```bash
rojo build -o "Grand Line Runner.rbxlx"
```

Next, open `Grand Line Runner.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

## Current Limitations
- `src` does not yet include all live place geometry, templates, and assets.
- Some runtime dependencies are still created dynamically or only exist in the live place file.
- Treat `build.rbxlx` as a verification artifact for code and Rojo-managed structure, not a guaranteed full-fidelity live export.

For more help, check out [the Rojo documentation](https://rojo.space/docs).
